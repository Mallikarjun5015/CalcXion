# CalcXion

> Precision. Speed. Power.  
> A modern interactive calculator built with React and designed with startup-grade UI aesthetics.

---

## Overview

CalcXion is a sleek, responsive calculator application built using React and Bootstrap.  
It focuses on delivering a smooth user experience with modern UI styling, tactile button animations, and sound feedback.

This project emphasizes:

- Clean component structure  
- Interactive UI experience  
- Responsive layout design  
- Modern visual aesthetics  
- Smooth press animations  

---

## Features

- Basic arithmetic operations
- Real-time expression rendering
- Instant result evaluation
- Button click sound feedback
- Modern light-themed interface
- Mobile responsive design
- Styled display screen section
- Button press animation effect

---

## Tech Stack

- React
- JavaScript (ES6+)
- Bootstrap
- CSS3
- HTML5
